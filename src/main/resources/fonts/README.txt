中文字体说明（用于 PDF 生成，避免乱码）
==========================================
将任意一种支持中文的字体文件放到本目录下，并命名为下列之一即可自动使用：
  - NotoSansCJKsc-Regular.otf  （推荐，Google Noto 思源黑体）
  - SimSun.ttf                 （Windows 宋体，可从系统复制）
  - msyh.ttf                   （微软雅黑）
  - SourceHanSansSC-Regular.otf

若本目录下没有上述字体，程序会尝试使用系统内置 STSongStd-Light；
若仍不可用，PDF 将使用默认字体，中文可能显示为方框或乱码。

Noto 字体下载示例：https://github.com/googlefonts/noto-cjk/releases
